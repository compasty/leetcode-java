# 概述

本项目为 [leetcode](https://leetcode.cn/) 练习和学习项目, 按照不同类型的知识点拆分为不同的包进行练习

具体来说:

1. **dp**: 动态规划
2. **tree**: 树（包含二叉树、二叉搜索树等）
3. **flip**: 滑动窗口
4. **list**: 链表
5. **stack**：堆栈
6. **backtrack**: 回溯
7. **sort**: 二分搜索，排序
